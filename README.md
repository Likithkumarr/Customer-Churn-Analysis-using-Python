## Customer Churn Analysis using Python
This project focuses on Exploratory Data Analysis (EDA) of a customer churn dataset to uncover insights about customer behavior and churn patterns. Using Python libraries such as Pandas, Matplotlib, and Seaborn, the analysis explores churn distribution, relationships between numerical and categorical features, detection of outliers, and correlation among variables.

## 📊 Project Overview
- Perform EDA on customer churn datasets.
- Visualize churn distribution across different features.
- Identify key drivers of churn using statistical and graphical methods.
- Highlight correlations between variables to support predictive modeling.

## 🛠️ Tools & Libraries
- Python 3
- Pandas – data manipulation and cleaning
- Matplotlib – visualization
- Seaborn – advanced statistical plots
- Jupyter Notebook – interactive analysis

## 📂 Repository Contents
- customer_chunk.ipynb – Jupyter Notebook with full EDA workflow.<br>
- customer_churn.csv – dataset used for analysis.<br>
- customer_churn_1000.csv – extended dataset for testing.<br>
- Customer_Churn_Project_PPT.pptx – presentation summarizing findings.<br>
- README.md – project documentation.<br>

## 🔍 Key Insights
- Distribution of churn across customer demographics and services.<br>
- Impact of numerical features (e.g., tenure, monthly charges) on churn.<br>
- Categorical feature analysis (e.g., contract type, payment method).<br>
- Outlier detection and handling strategies.<br>
- Correlation heatmaps to identify strong relationships.<br>
